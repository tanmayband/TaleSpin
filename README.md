# TaleSpin

Spin your tales your way.

(Yeah yeah, this project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app). Grow up.)

## Quick setup
### Client

1. `cd client`
2. `npm install`
3. `npm start`

### Server

1. `cd server`
2. `npm install`
3. `npm install -g nodemon`
3. `nodemon server`
